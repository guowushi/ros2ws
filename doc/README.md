


# 界面设计

## 左侧 (200px)
包含14 个控制按钮：
- 回原点
- 急停
- 扫描控制 (4)、
- 打磨控制 (3)
- 辅助操作 (4)

### 按钮
- 启动：启动软件，准备工作。此时会通过ModbusTCP连接PLC，连接相机；启动其他相关后端服务。在状态栏分别显示“PLC连接状态”，“相机连接状态”，“后端服务的状态”，绿色表示正常，红色表示异常。

- 开始扫描：在“相机状态”正常时，允许点击“开始扫描”按钮。扫描结束后

## 中间 (自适应) 
中间 (自适应) │ Qt Quick 3D 场景 — 6 轴机器人关节臂模型 + 参考网格面 + 方向光 
## 右侧 (260px)
使用多个TAB页面分别显示不同的参数：

### TAB1 
 机器人基础信息，包括：
 - 关节角度 A1–A6
 - 笛卡尔坐标 XYZ ABC、
 - 外部轴：位移、使能、到位、运动

 
 
### TAB2
 工艺参数：打磨压力、打磨速度、行走速度
 路径：总路径条数、当前路径编号、当前路径总点数、已接收点数

## TAB3
HMI信息：
- 机器人回原点
- 快换打开
- 快换关闭
- 导轨回原点
- 相机气抓



## 底部状态栏  
分别包含相关状态：PLC状态、相机状态、后端服务状态
还包括当前参数：工件编号、时间。
